# Spreadsheet
Spreadsheet that can manage formulas and several functions 
in order to perform some calculations.

### Features available
- Add text to a cell
- Add numbers to a cell
- Compute a formula inside a cell
- Usage of functions inside a formula such as:
  - SUM
  - MIN
  - MAX
  - AVERAGE
